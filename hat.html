Henry's Animation Tool

I originally created Henry's Animation Tool as a way to create movies in Garry's Mod.

I was only 16 at the time that I started work on Henry's Animation Tool and the only thing I knew at that time was programming.

I loved creating movies in video games and sharing them with my friends, but creating movies is very tedious and requires a lot of hard work and patience.

Using my skills, I created an animation tool that I, or any other layman, could use without a lot knowledge in animation.

When it was released, I had my 15 minutes of fame as my name was broadcasted through a handful of gaming blogs and my tool being showcased on the main Garry's Mod page.

Since it's initial release, Henry's Animation Tool has gone through many many changes, but it kept it's main focus to "keep it simple".

It's accumulated over 13,000 subscribers on Steam Workshop.
